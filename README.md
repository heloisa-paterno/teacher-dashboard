# Teacher Dashboard on Student Performance

## Goal
Present and visualize data on homework, tests, and classroom activity performance across classes.

## Tools
Looker Studio, Google Sheets, Python

## Dataset
Sample dataset on student data.

## Features
- Cross-class comparison
- Best test score calculation
- Filters by class/year
- More

## Insights
Students and classes with low homework completion had worse test improvement.
